## Requirements

1. neovim
    - `<version> > 0.8.0`
2. git
3. make

### neovim

```bash
# macos -- please use brew install
brew install neovim

# ubuntu
apt-get update && apt-get install neovim
```

## How to install hwloser-vim

```bash
# install this manager
make install

# uninstall this manager
make uninstall
```
